# S3Manager-Laravel

Projeto para controle de arquivos no S3

## Recursos

- [Laravel](https://laravel.com/)
- [Vue](https://vuejs.org/)
- [Buefy](https://buefy.org/)
- [Bulma](https://bulma.io/)

## Configuração

Seu arquivo `.env` deve conter os seguintes dados do seu repositório S3.

```jsx
AWS_ACCESS_KEY_ID=
AWS_SECRET_ACCESS_KEY=
AWS_DEFAULT_REGION=
AWS_BUCKET=
AWS_FOLDER_ROOT= 
```
